# 📚 Materi Praktikum Komputer Vision

## Berdasarkan Buku: Computer Vision - Algorithms and Applications (2nd Edition)
### Oleh: Richard Szeliski

---

## 📂 Struktur Folder

Materi ini disusun berdasarkan bab-bab dalam buku Computer Vision karya Richard Szeliski. Setiap folder berisi:

1. **`materi.md`** - Materi teori lengkap dalam Bahasa Indonesia
2. **`praktikum/`** - Subfolder berisi kode praktikum dan latihan
3. **`tugas/`** - Subfolder berisi soal-soal tugas
4. **`referensi.md`** - Referensi tambahan dan bacaan lanjutan

## 🎯 Tujuan Pembelajaran

Setelah mempelajari materi ini, mahasiswa diharapkan mampu:

1. **Memahami** konsep dasar computer vision dan aplikasinya
2. **Mengimplementasikan** algoritma-algoritma computer vision menggunakan Python
3. **Menganalisis** permasalahan visual dan memilih solusi yang tepat
4. **Mengembangkan** aplikasi berbasis computer vision

---

## 🛠️ Tools yang Digunakan

- **Python 3.8+**
- **OpenCV** - Library utama untuk computer vision
- **NumPy** - Operasi array dan matematika
- **Matplotlib** - Visualisasi
- **scikit-image** - Algoritma pemrosesan citra
- **TensorFlow/PyTorch** - Deep learning
- **Jupyter Notebook** - Untuk praktikum interaktif

## 📖 Cara Menggunakan Materi

1. Baca materi teori (`materi.md`) terlebih dahulu
2. Pelajari contoh kode di folder `praktikum/`
3. Kerjakan latihan yang ada
4. Selesaikan tugas di folder `tugas/`
5. Baca referensi tambahan untuk pemahaman lebih dalam
