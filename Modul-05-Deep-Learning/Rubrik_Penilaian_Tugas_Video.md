## D. Bonus dan Penalti

### Bonus

| Aspek | Bonus |
|-------|-------|
| **Subtitle** | +5% untuk subtitle Bahasa Indonesia |
| **Diagram Original** | +3% untuk diagram/animasi buatan sendiri |
| **Real-world Application** | +5% untuk demo dengan aplikasi nyata |
| **Comparative Analysis** | +3% untuk perbandingan framework/model yang mendalam |
| **Live Webcam Demo** | +5% untuk demo real-time yang smooth |

### Penalti

| Aspek | Penalti |
|-------|---------|
| **Terlambat** | -10% per hari keterlambatan (max 3 hari) |
| **Plagiat** | -50% hingga 0 untuk konten plagiat |
| **Tidak Original** | -20% untuk konten copy dari tutorial tanpa modifikasi |
| **Error Tidak Dihandle** | -5% untuk error yang dibiarkan tanpa penjelasan |
| **Format Salah** | -5% untuk format file selain yang diminta |

## Perhitungan Nilai Akhir

### Formula

```
Nilai Video = A + B + C + Bonus - Penalti

Dimana:
A = Skor Konten (max 50%)
B = Skor Demonstrasi Praktis (max 30%)
C = Skor Kualitas Produksi (max 20%)
Bonus = Max 21%
Penalti = Varies
```
