# Rubrik Penilaian Tugas Video
## Bab 2: Pembentukan Citra (Image Formation)

---

## 🎬 Deskripsi Tugas

Buat video tutorial/demonstrasi dengan durasi **8-12 menit** yang menjelaskan dan mendemonstrasikan konsep **Transformasi Geometri dan Kalibrasi Kamera** dalam Computer Vision.

---

## 📋 Checklist Konten Video

### Bagian 1: Pembukaan (1-2 menit)
- [ ] Perkenalan diri (nama, NIM)
- [ ] Judul topik yang akan dibahas
- [ ] Overview singkat apa yang akan dipelajari
- [ ] Hook menarik mengapa topik ini penting

### Bagian 2: Teori Dasar (2-3 menit)
- [ ] Penjelasan konsep transformasi geometri
- [ ] Perbedaan translation, rotation, scaling
- [ ] Penjelasan affine vs perspective transform
- [ ] Visualisasi/animasi transformation matrix
- [ ] Penjelasan camera model (pinhole, lens distortion)
- [ ] Sampling & aliasing (ringkas)
- [ ] Color space & gamma correction (ringkas)

### Bagian 3: Demonstrasi Kode (3-4 menit)
- [ ] Setup environment dan import library
- [ ] Demo transformasi dasar (translasi, rotasi, scaling)
- [ ] Demo transformasi affine
- [ ] Demo transformasi perspektif
- [ ] Penjelasan fungsi OpenCV yang digunakan
- [ ] Menunjukkan efek perubahan parameter
- [ ] (Opsional) Demo color space / gamma / aliasing

### Bagian 4: Aplikasi Nyata (1-2 menit)
- [ ] Demo document scanner sederhana
- [ ] Contoh aplikasi di industri (CamScanner, AR, dll)
- [ ] Hubungan dengan kalibrasi kamera
- [ ] Contoh dampak kompresi/gamma pada kualitas citra

### Bagian 5: Penutup (1 menit)
- [ ] Ringkasan poin-poin utama
- [ ] Tips dan best practices
- [ ] Referensi untuk belajar lebih lanjut
- [ ] Call to action (ajakan mencoba sendiri)

## 📌 Spesifikasi Teknis Video

### Minimum Requirements
- **Durasi**: 8-12 menit (toleransi ±1 menit)
- **Resolusi**: Minimal 720p (1280×720)
- **Format**: MP4, MOV, atau link YouTube/Google Drive
- **Audio**: Jelas, minimal noise, volume konsisten

### Recommended
- **Resolusi**: 1080p (1920×1080)
- **Frame Rate**: 30fps atau 60fps
- **Screen Recording**: OBS Studio, Camtasia, atau sejenisnya
- **Editing**: Basic cuts dan transitions
- **Thumbnail**: Custom thumbnail yang menarik

### Submission Format
1. Upload ke YouTube (Unlisted) atau Google Drive
2. Submit link melalui LMS
3. Pastikan link dapat diakses oleh penilai
4. Sertakan file kode yang didemonstrasikan

## 💡 Tips Membuat Video yang Baik

### Persiapan
1. **Buat script/outline** - Jangan improvisasi total
2. **Test audio terlebih dahulu** - Pastikan mikrofon berfungsi baik
3. **Siapkan semua kode** - Test semua demo sebelum recording
4. **Bersihkan desktop** - Hindari distraksi visual

### Recording
1. **Bicara dengan jelas dan tidak terlalu cepat**
2. **Zoom ke kode** saat menjelaskan detail
3. **Pause sebelum transisi** untuk memudahkan editing
4. **Jangan takut melakukan retake** jika ada kesalahan

### Editing
1. **Potong bagian yang tidak perlu** (loading, typo, dll)
2. **Tambahkan text/annotation** untuk highlight poin penting
3. **Gunakan zoom effect** untuk menekankan kode
4. **Pastikan audio level konsisten**

### Konten
1. **Mulai dengan "mengapa"** sebelum "bagaimana"
2. **Gunakan analogi** untuk konsep yang kompleks
3. **Tunjukkan error dan cara mengatasinya** - sangat edukatif
4. **Hubungkan dengan aplikasi real-world**
