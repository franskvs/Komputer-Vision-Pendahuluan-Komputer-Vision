## Deskripsi Tugas

Mahasiswa membuat video penjelasan tentang salah satu topik recognition:
1. Face Detection dan Face Recognition
2. Object Classification dengan CNN
3. Scene Recognition dan Attributes
4. OCR dan Text Recognition

Video harus mencakup:
- Penjelasan konsep teori
- Demonstrasi implementasi
- Analisis hasil dan aplikasi

## Tips untuk Video Berkualitas

### Persiapan:
1. Buat script/outline terlebih dahulu
2. Test recording setup (audio, screen)
3. Siapkan code dan demo sebelum recording
4. Practice run sekali sebelum recording

### Recording:
1. Gunakan mikrofon eksternal jika ada
2. Minimize background noise
3. Berbicara dengan pace steady
4. Pause di transisi penting

### Post-production:
1. Edit bagian yang error
2. Tambahkan zoom-in pada code penting
3. Sertakan subtitle jika possible
4. Review sebelum submit
