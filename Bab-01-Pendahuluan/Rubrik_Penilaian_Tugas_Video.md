# RUBRIK PENILAIAN TUGAS VIDEO
## BAB 1: PENDAHULUAN COMPUTER VISION

---

## 📊 RUBRIK PENILAIAN DETAIL

### TOTAL NILAI: 100 POIN

---

### 1. PEMBUKAAN (10 Poin)

| Skor | Kriteria |
|------|----------|
| **9-10** (Excellent) | Perkenalan lengkap, menyebutkan judul materi, tujuan pembelajaran jelas, overview menarik |
| **7-8** (Good) | Perkenalan cukup lengkap, ada judul dan tujuan |
| **5-6** (Fair) | Perkenalan ada tapi kurang lengkap |
| **3-4** (Poor) | Perkenalan minimal, tidak ada tujuan |
| **0-2** (Fail) | Tidak ada pembukaan / langsung ke materi |

**Checklist Pembukaan:**
- [ ] Menyebutkan nama dan identitas (2 poin)
- [ ] Menyebutkan judul materi (2 poin)
- [ ] Menyebutkan tujuan pembelajaran (3 poin)
- [ ] Overview singkat konten video (3 poin)

---

### 2. PENJELASAN MATERI (25 Poin)

| Skor | Kriteria |
|------|----------|
| **23-25** (Excellent) | Definisi CV dijelaskan dengan bahasa sendiri, diagram pipeline jelas, contoh aplikasi relevan dan menarik |
| **18-22** (Good) | Penjelasan baik dan mudah dipahami, ada visual aids |
| **13-17** (Fair) | Penjelasan cukup, tapi ada konsep yang kurang jelas |
| **8-12** (Poor) | Penjelasan minimal, banyak membaca teks |
| **0-7** (Fail) | Tidak ada penjelasan materi / salah konsep |

**Checklist Materi:**
- [ ] Definisi Computer Vision dengan bahasa sendiri (5 poin)
- [ ] Diagram pipeline CV (bisa digambar/slides) (5 poin)
- [ ] Penjelasan level vision (Low/Mid/High) (5 poin)
- [ ] Minimal 3 contoh aplikasi nyata di kehidupan (5 poin)
- [ ] Penjelasan representasi gambar digital (5 poin)

---

### 3. PRAKTIKUM & DEMONSTRASI (35 Poin)

#### 3.1 Setup & Environment (5 Poin)
| Skor | Kriteria |
|------|----------|
| **5** | Menunjukkan environment, menjelaskan instalasi, siap didemonstrasikan |
| **3-4** | Environment terlihat tapi tidak dijelaskan |
| **1-2** | Setup tidak lengkap |
| **0** | Tidak menunjukkan setup |

#### 3.2 Percobaan 1 - Loading Gambar (5 Poin)
| Skor | Kriteria |
|------|----------|
| **5** | Demo berjalan, kode dijelaskan baris per baris, hasil ditunjukkan |
| **3-4** | Demo berjalan, penjelasan cukup |
| **1-2** | Demo berjalan tapi tidak ada penjelasan |
| **0** | Tidak didemonstrasikan |

#### 3.3 Percobaan 2 - Properti Gambar (5 Poin)
| Skor | Kriteria |
|------|----------|
| **5** | Menjelaskan setiap properti (dimensi, channel, dtype), analisis hasil |
| **3-4** | Menunjukkan output, penjelasan cukup |
| **1-2** | Hanya menunjukkan tanpa penjelasan |
| **0** | Tidak didemonstrasikan |

#### 3.4 Percobaan 3 - Konversi Warna (5 Poin)
| Skor | Kriteria |
|------|----------|
| **5** | Demo semua konversi (RGB, Grayscale, HSV), menjelaskan perbedaan visual |
| **3-4** | Demo berfungsi, penjelasan cukup |
| **1-2** | Hanya sebagian konversi |
| **0** | Tidak didemonstrasikan |

#### 3.5 Percobaan 4 - Manipulasi Piksel (5 Poin)
| Skor | Kriteria |
|------|----------|
| **5** | Menjelaskan akses piksel, modifikasi region, hasil terlihat jelas |
| **3-4** | Demo berfungsi, ada penjelasan |
| **1-2** | Demo tanpa penjelasan |
| **0** | Tidak didemonstrasikan |

#### 3.6 Percobaan 5 - Drawing Shapes (5 Poin)
| Skor | Kriteria |
|------|----------|
| **5** | Demo semua shape (line, rectangle, circle, text), parameter dijelaskan |
| **3-4** | Demo berfungsi, beberapa shape |
| **1-2** | Hanya 1-2 shape |
| **0** | Tidak didemonstrasikan |

#### 3.7 Percobaan 6 - Menyimpan Output (5 Poin)
| Skor | Kriteria |
|------|----------|
| **5** | Demo save berbagai format, perbandingan ukuran file, kualitas |
| **3-4** | Demo save berfungsi, ada penjelasan |
| **1-2** | Hanya save tanpa penjelasan |
| **0** | Tidak didemonstrasikan |

---

### 4. ANALISIS HASIL (15 Poin)

| Skor | Kriteria |
|------|----------|
| **14-15** (Excellent) | Analisis mendalam setiap percobaan, perbandingan hasil, insight yang bagus |
| **11-13** (Good) | Analisis cukup baik, ada perbandingan |
| **8-10** (Fair) | Analisis dasar ada |
| **4-7** (Poor) | Analisis minimal |
| **0-3** (Fail) | Tidak ada analisis |

**Checklist Analisis:**
- [ ] Menunjukkan output setiap percobaan (5 poin)
- [ ] Membandingkan hasil dengan ekspektasi (5 poin)
- [ ] Menyebutkan pembelajaran dari masing-masing percobaan (5 poin)

---

### 5. PENUTUP (10 Poin)

| Skor | Kriteria |
|------|----------|
| **9-10** (Excellent) | Rangkuman poin penting, kesulitan & solusi, kesimpulan reflektif, salam penutup |
| **7-8** (Good) | Rangkuman dan kesimpulan baik |
| **5-6** (Fair) | Kesimpulan ada tapi kurang detail |
| **3-4** (Poor) | Kesimpulan minimal |
| **0-2** (Fail) | Tidak ada penutup |

**Checklist Penutup:**
- [ ] Rangkuman poin-poin penting (3 poin)
- [ ] Menyebutkan kesulitan yang dihadapi & solusi (2 poin)
- [ ] Kesimpulan pembelajaran (3 poin)
- [ ] Salam penutup (2 poin)

---

### 6. KUALITAS VIDEO & AUDIO (5 Poin)

| Skor | Kriteria |
|------|----------|
| **5** (Excellent) | Video jernih 720p+, audio jelas, tidak ada noise, pencahayaan baik |
| **4** (Good) | Video cukup jernih, audio dapat didengar |
| **3** (Fair) | Video atau audio kurang baik tapi masih acceptable |
| **2** (Poor) | Kualitas video/audio buruk, sulit dipahami |
| **0-1** (Fail) | Tidak dapat dilihat/didengar dengan baik |
